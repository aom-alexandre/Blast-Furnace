cc - carbon counter
oc - oxygen counter
sic - silicon counter
cac - calcium counter
fec - iron counter

co2 - if carbon dioxide exists
co1 - if carbon monoxide exists
casio3 - if slag exists
c - if carbon exists
fe2o3 - if hematite exists
oo - if oxygen molecule exists
caco - if CaCO3 exists
cao - if calcium oxide exists
fe - if iron exists
sio2 - if SiO2 exists

cor - co2+c->co reaction counter
cr - c+o2->co2 reaction counter
hematite - reaction of hematite with carbon monoxide counter
slag - reaction to form slag counter
caor - reaction that creates cao counter

place - number of things spawned
p - second so that the image doesn't stay on the same space
placed - number of things placed correctly
x, y - coordinates of things
